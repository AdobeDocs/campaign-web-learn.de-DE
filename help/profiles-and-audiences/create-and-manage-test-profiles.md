# Erstellen und Verwalten von Testprofilen

Erfahren Sie, wie Sie ein Testprofil in der Client-Konsole erstellen und wie Sie Profile in der Adobe Campaign-Web-Benutzeroberfläche verwalten und bearbeiten.
